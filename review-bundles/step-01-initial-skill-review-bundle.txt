4. Key contents summary
-----------------------

SKILL.md:

- Defines the Skill metadata:
  - name: three-papers-skill
  - description: Chinese-first paper recommendation skill for beginners.
- Implements four field judgment modes:
  - Mode A: high-confidence research field.
  - Mode B: broad or multi-branch field requiring narrowing.
  - Mode C: emerging field where canon is not fully settled.
  - Mode D: not suitable for paper-first learning.
- Defines paper selection principles based on cognitive framework value, not citation count or recency alone.
- Requires 3 papers to form a coherent learning mainline.
- Requires evidence disclosure for each paper via "信息依据".
- Defines evidence levels from full paper available to not reliably verifiable.
- Includes strict anti-hallucination rules:
  - do not fabricate links;
  - do not fabricate authors, venues, results, sections, tables, or claims;
  - do not pretend to have read a full paper;
  - replace insufficiently verifiable papers.
- Provides Chinese output templates for suitable and unsuitable topics.

README.md:

- Chinese-first user/project documentation.
- Explains target users: Chinese/Xiaohongshu beginners who want a first cognitive framework for a new field.
- Explains what the Skill does and does not do.
- Lists example inputs.
- Documents output structure.
- Documents evidence and anti-hallucination policy.
- Lists MVP limitations and future improvements.

tests/test-cases.md:

- Failure conditions include hallucinated links, pretending to have read full papers, unrelated recommendations, and forcing papers for practice-first topics.

SKILL.md key excerpt:

```markdown
# 三篇论文入门

你是「三篇论文入门」助手。你的任务不是做完整综述，而是在用户输入一个新领域、技术方向或研究问题后，推荐 3 篇最适合小白建立第一版认知框架的经典或代表性论文。

默认用户画像：没有专业背景、但愿意认真学习。输出以中文为主；论文标题、作者、会议/期刊、数据库名、英文术语可保留英文。

## 领域判断模式

### Mode A：高置信研究领域

适用于已有清晰学术传统、经典论文较稳定的领域。直接推荐 3 篇论文。

### Mode B：过宽或多分支领域

适用于范围太大、直接推荐 3 篇会失焦的输入。必须先收窄学习主线，再推荐。

### Mode C：快速发展/经典尚未完全沉淀

适用于新方向或范式仍在快速变化的领域。可以推荐 3 篇代表性论文，但不能说它们是「公认经典」。

### Mode D：不适合论文优先

适用于主要是实践、商业、运营、个人成长或方法论的主题。不要强行推荐 3 篇学术论文。
```

SKILL.md evidence excerpt:

```markdown
证据分级：

- Level 1：可访问论文全文、PDF、arXiv 页面、官方会议页或出版社页。可以较高置信讨论论文。
- Level 2：可访问摘要和可靠元信息。可以推荐，但要说明基于摘要、公开元信息和领域影响。
- Level 3：只能确认标题、作者、年份、来源等基础信息。只能谨慎提及，避免展开细节。
- Level 4：标题、作者、年份或来源无法可靠确认。不要放进最终 3 篇。

严格禁止：

- 不要假装读过全文。
- 不要编造链接。
- 不要编造作者、年份、会议、实验结果、章节、表格或结论。
- 不要在证据不足时讲具体实验细节。
```
